Virtual Evil

Invade yer space

Java 2D space invaders game. I wanted to experiment with user experience: animations, notifications and sounds. I also encountered concurrent data structures and had to reason about algorithmic complexity of object collision detection.

Sanity's Eclipse

This one is a Java 2D asteroids-like game. We learned about using Java in a realtime application, threading, basic enemy AI and some trigonometry tricks. I worked with Bozhidar Hrusanov.

Strength and Steel

2D tiled fantasy game written in C++ with Allegro 4 (because it was 2012). It taught me OOP, rendering 2D graphics and using arrays of pointers to npcs.

A 3D shooter using Doom sprites. Building this game allowed me to apply knowledge in trigonometry and stereometry. Made with GM7 and Blender. There are 9 levels with 2 bosses, 4 weapons, 6 basic enemies.
